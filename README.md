# OpenApex 
 
OpenApex is a personal assistant framework developed by Afriyono Dwi Saputro (Bapak Afri_kenyik). 
It provides tools for AI-driven automation, integration with GitHub, system commands, and more. 
 
## Getting Started 
 
1. Clone the repository 
   ```bash 
   git clone https://github.com/growup62/openapex.git 
   cd openapex 
   ``` 
2. Install dependencies 
   ```bash 
   pip install -r requirements.txt 
   ``` 
3. Run the core assistant 
   ```bash 
   python main.py 
   ``` 
 
## Features 
 
- System command execution 
- Web search integration 
- GitHub issue/PR management 
- Physical mouse/keyboard control 
- Long-term memory with ontology
